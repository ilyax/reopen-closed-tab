# Contributing

Thanks for contributing to Reopen Closed Tab.

## Development Setup

1. Clone the repository.
2. Open `chrome://extensions`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the project folder.

## Making Changes

- Keep the implementation small and dependency-free.
- Prefer simple, readable code over abstraction.
- Update documentation when behavior changes.
- Test changes manually by reloading the extension and reopening a recently closed tab or window.

## Pull Requests

- Use clear commit messages.
- Keep pull requests focused.
- Describe the user-visible behavior change.
- Include any manual testing notes in the pull request description.

## Issues

If you find a bug or want to propose an enhancement, open an issue with reproduction steps or a clear problem statement.
