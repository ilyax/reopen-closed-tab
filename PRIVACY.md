# Privacy Policy

## Summary

Reopen Closed Tab does not collect, store, share, or transmit personal data.

## Data Usage

- No analytics
- No telemetry
- No cookies
- No external API calls
- No remote code execution

## Permissions

The extension requests the following permission:

- `sessions`: Used only to access Chrome's recently closed session list and restore the latest closed tab or window.
